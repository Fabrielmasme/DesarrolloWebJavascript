Simulador de Tienda - Entregable 1

Este simulador permite a un usuario registrar productos en un carrito, calcular totales y ver un resumen. La interacción principal se realiza mediante la Consola JS y cuadros de diálogo (prompt / confirm / alert).

El objetivo es integrar variables, arrays, objetos, funciones, condicionales y bucles. El HTML sólo contiene la descripción y el enlace para iniciar el simulador (la lógica está en el archivo JS).
